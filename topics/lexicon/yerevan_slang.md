# Yerevan slang and jargon

Yerevan colloquial speech contains a substantial layer of **jargon /
slang vocabulary** (Armenian: *ժարգոն* / ժարգոնային բառեր) — youth-
register lexical items absent from the literary norm. Ghamoyan
documents this as a defining feature of Yerevan colloquial speech,
"often used to be incomprehensible to others" historically tied to
sub-cultures but now widespread. [#1]

## High-frequency slang

A sampler from ghamoyan p48 [#1]:

| Yerevan slang | Russian gloss / meaning | English |
|---------------|------------------------|---------|
| բոց ա | (col.) | "(it's) cool / amazing" — lit. "(it's) flame" |
| թույն (ա) | хорошо | "cool / great" — lit. "poison" |
| քյառթու | вульгарный | "vulgar / loutish" |
| փոստ ա | развлекательный | "(it's) entertaining" |
| լոքշ ա | безделье | "(it's) idleness / lameness" |
| ֆազոտ | неустойчивый человек | "unstable / sketchy person" |
| ֆռցնել | обмануть | "to deceive / scam" |
| ցրել | украсть | "to steal" — lit. "to scatter" |
| բքել | курить много | "to chain-smoke" |
| դուխով | смело, со смелостью | "with spirit / brave" |
| դզող | (col.) | "cool" — lit. "fitting" |
| գողական | (col.) | "thuggish/criminal-style" |
| գազան կինո | (col.) | "awesome movie" |

`թույն` "cool" is particularly productive: `թույն կինո` "cool
movie", `թույն աղջիկ` "cool girl", `թույն մաշնա` "cool car",
`թույն շոր` "cool clothes." Combinable with most positive-evaluation
nouns. [#1]

## How slang is formed

Per ghamoyan, two main patterns: [#2]

**Pattern A: existing words get new (often metaphorical) meanings.**
A literary-register word gets repurposed for a slang sense, often
unrelated to its original meaning. Examples [#2]:

| word | literary meaning | slang meaning |
|------|------------------|---------------|
| խոտ | "grass" | "naive / clueless person" |
| ծախել | "to sell" | "to betray, to snitch" |
| ծախվել | "to be sold" | "to be betrayed; to give in" |
| ծակվել | "to be pierced" | "to accidentally reveal the truth" |
| կերցնել | (causative of "eat") | "to give the answer" (cheating) |
| կոշկվել | (passive of "shoe") | "to land in a tough spot" |
| կոտրել | "to break" | "to discredit / ruin reputation" |
| չորել | "to dry" | "to be rude" |
| քաշել | "to pull / drag" | "to harm" |

**Pattern B: foreign-word borrowing** — Russian borrowing predominates.
This is documented in
`topics/lexicon/code_switching_with_russian.md`.

## Abbreviations / clipped forms

A separate jargon pattern is **clipping** of long words to short
forms used in school/university student speech: [#1]

| full | clipped |
|------|---------|
| անգլերեն "English" | անգլ |
| աշխարհագրություն "geography" | աշխարհ |
| բնագիտություն "natural science" | բնագ |
| թվաբանություն "arithmetic" | թվաբ |
| պատմություն "history" | պատմ |
| ժամադրություն "appointment" | ժամո |
| մագնիտոֆոն "tape recorder" | մագ |
| ծածկաթերթիկ "cheat sheet" | շպո |
| լիկվիդացիա "liquidation" | լիկվիդ |

Pattern: keep the first 1-2 syllables, drop the rest. This is
universal across Yerevan student speech and unmarked for register
within that population.

## Pragmatic notes

- **Don't deploy slang as a beginner.** Most of these items are
  register-marked; using them in a formal context (job interview,
  introduction to elders) would be jarring at best, offensive at
  worst. Useful to *understand* in dialogue; deploy only after
  confirming the social register matches.
- **Some items are gendered or class-marked.** ghamoyan notes that
  some jargon used to be male-only but is now used by women too;
  the social signalling of slang shifts faster than its lexical
  inventory.
- **Slang ages quickly**, especially among teens. ghamoyan's 2014
  sample may already be partially dated — `բոց` and `թույն` for
  "cool" remain current; some of the 2010s items may have been
  replaced.

## Contrastive notes

**For an English L1**: think of these as the Armenian equivalents
of "lit", "sick", "cap", "based", "fire" — register markers for
peer-group speech that shift every few years. Don't try to
translate lexically; map function-to-function. `թույն ա` ≈ "it's
sick / it's fire."

**For a Russian L1**: many of these are direct Russian-influenced
patterns or Russian borrowings (see code-switching topic). The
metaphorical-extension pattern (Pattern A above — `խոտ` "grass"
→ "naive person") is a calque of the Russian/world-Slavic slang
pattern (`лох`, `чайник`, etc.). The cognitive template is closer
to Russian than to English.
